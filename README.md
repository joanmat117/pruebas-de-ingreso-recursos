# Recursos para pruebas de ingreso — Pruebas, resúmenes y ejercicios

**Descripción**
Repositorio con materiales de apoyo para pruebas de ingreso en Cuba: resúmenes, ejercicios, pruebas de ejemplo y respuestas. El objetivo es reunir recursos organizados por materia y nivel para facilitar el estudio y la preparación.

Repositorio: [joanmat117/pruebas-de-ingreso-recursos](https://github.com/joanmat117/pruebas-de-ingreso-recursos)

---

## Contenido
- Materiales de estudio (resúmenes y apuntes).
- Ejercicios y pruebas tipo examen.
- Soluciones y guías de corrección.
